# Smart Civic Complaint Management System

A full-stack web application that allows users to report civic issues (road damage, garbage, water leakage) and track their complaint status in real time.

## 🚀 Features
- Submit complaints with image upload 📸
- Auto location detection 📍
- Unique complaint ID generation 🆔
- Track complaint status 🔍
- Admin dashboard to manage complaints 👨‍💼
- Real-time status updates ⏱️

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Tools: Multer (file upload)
